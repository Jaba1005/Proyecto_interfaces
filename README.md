De lo que pide el profe ya esta haceho:
  1-Verifica que la aplicación controle el acceso de usuarios no 
    registrados, gestione el inicio de sesión de forma segura e identifique 
    correctamente al usuario activo

  2-Evalúa que la aplicación mantenga la sesión activa durante toda la 
    navegación, conserve los datos del usuario logueado y ofrezca la 
    opción de cerrar sesión de manera correcta, bloqueando el acceso al 
    intentar volver con la flecha de retroceso 

  3-Implementa menús adaptados según el rol o perfil del usuario en 
    sesión - DE MOMENTO SOLO ESTA ADMIN Y USUARIO NO SE SI QUIERES AGREGAR ALGUN OTRO ROL

  4-La app permite realizar operaciones de crear, leer, actualizar y 
   eliminar datos generadas de manera integrada y funcional desde una 
   sola vista o interfaz, garantizando la correcta conexión y 
   sincronización con la base de datos externa (MySQL). 

  5-La gestión total del sistema debe realizarse a través de las interfaces 
   gráficas. Se evalúa la usabilidad, consistencia visual, validación de 
   campos vacíos y coherencia entre los formularios y las operaciones 
   administrativas.
